# API Biblioteca Avançada

API REST desenvolvida com **Node.js, Express, Prisma e PostgreSQL**, seguindo arquitetura **MVC**.

## Funcionalidades

- CRUD completo de **Autores** e **Livros**
- Relação entre autores e livros
- **Paginação**, **ordenação** e **pesquisa**
- Endpoint de **estatísticas**
- **Validação de dados**
- **Seed da base de dados**
- **Middleware global de erro**
- Deploy na Vercel
